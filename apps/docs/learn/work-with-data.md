# Work with your data

Datasets turn Aglyn from a page builder into a content platform. This path takes
you from a data model to dynamic pages — and, optionally, the API.

1. **[Datasets & dynamic content](/content-and-data/datasets/overview)** — the
   big picture: typed models, records, relations, and repeatables.
2. **[Build a data model](/content-and-data/datasets/model-builder)** — define
   fields and edit records.
3. **[Relations](/content-and-data/datasets/relations)** — link records together.
4. **[Datasets & schema deep-dive](/guides/datasets-and-schema)** — field ids vs
   display names, quotas, and every writer that can append records.
5. **[Bindings, variables & functions](/building-sites/bindings/overview)** — pull
   live values into your content.
6. **[Read & write over the API](/api/resources/datasets)** — manage records
   programmatically (Business plan).
