# Multilingual

Reach a wider audience by publishing your site in more than one language.

:::info Plan availability
**Paid**. Multilingual is available on paid tiers.
:::

![The Languages card in site setup](/img/multilingual/setup-languages.png)

## Locale variants

Create **locale variants** of your content so each language has its own version, while
sharing the same structure.

## hreflang & discovery

Aglyn emits **hreflang** tags so search engines serve the right language to the right
visitor.

## Language switcher

Add a **language switcher** so visitors can move between locales themselves.

:::tip How-tos
- [Add a locale](add-a-locale.md)
- [Add a language switcher](language-switcher.md)

This is **multilingual v1**.
:::
